# 12-App Quick Launcher (12宫悬浮球)

> **极简、高效、无隐私侵犯。**

## 💡 开发初衷
市面上的悬浮球应用大多存在两个极端：要么功能过于臃肿，塞入了一堆用不上的功能；要么为了实现“模拟点击”而强制要求开启 **“无障碍服务 (Accessibility Service)”** 权限，这带来了隐私泄露的风险。

而系统自带的悬浮球（如小米悬浮球）往往限制只能添加 5 个应用，对于重度手机用户来说捉襟见肘。

**12-App Quick Launcher** 因此诞生。它专注于**纯粹的快速启动**，无需敏感权限，轻量且高效。

---

## ✨ 核心功能
* **🧩 12宫格菜单**：突破系统限制，支持自定义 12 个常用应用，一键触达。
* **🛡️ OLED 防烧屏保护**：独家算法，在隐藏模式下每分钟自动进行像素级微小位移，防止悬浮球长期静止导致屏幕烧屏。
* **🌗 双色高对比设计**：采用独特的“白芯黑边”双层圆环设计（LayerDrawable），无论是纯白背景还是深色夜间模式，都能清晰识别，拒绝“隐身”。
* **🤖 智能吸附与避让**：
    * 屏幕旋转后，自动计算并吸附到侧边垂直居中位置。
    * **强制转角避让**：禁止悬浮球停留在屏幕四个死角，防止误触或难以拖出。
    * **3秒自动隐藏**：无操作 3 秒后自动半透明贴边隐藏。
* **👁️ 全分辨率适配**：内置 DP 转 PX 算法，完美适配 1K / 2K / 4K 等各种分辨率屏幕，图标大小始终适中。

## ⚠️ 使用注意事项与局限性
由于本项目坚持**不申请系统级敏感权限**（如 Root 或 无障碍服务），受限于安卓系统机制，存在以下特性：

1.  **系统版本**：建议使用 **Android 9.0 (Pie)** 及以上系统以获得最佳体验（支持刘海屏/挖孔屏区域显示）。
2.  **截图可见**：因为不是系统级应用，无法在截图时自动隐藏，悬浮球会被包含在截图中。
3.  **常驻通知**：系统通知栏可能会提示“正在其他应用的上层显示内容”。这是安卓对悬浮窗应用的安全提示，您可以长按该通知并关闭此类通知显示，不影响使用。
4.  **保活设置（重要）**：为了防止被系统后台查杀导致消失，请务必：
    * 将本应用的电池策略设置为 **“无限制”**。
    * 在多任务界面将本应用 **“锁定”**。
